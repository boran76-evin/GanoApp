GanoApp (GANO Hesaplama Uygulaması)

Bu proje, üniversite öğrencilerinin derslerini ekleyip notlarına göre **GANO (Genel Ağırlıklı Not Ortalaması)** hesaplayabildiği bir mobil uygulamadır.

## Özellikler

- Ders ekleme
- Ders düzenleme
- Ders silme
- Harf notuna göre GANO hesaplama
- Ortalamaya dahil / hariç seçimi
- Modern ve kullanıcı dostu arayüz
- SQLite veritabanı kullanımı

## Kullanılan Teknolojiler

- .NET MAUI
- C#
- SQLite
- XAML

## Nasıl Çalışır?

- Kullanıcı ders bilgilerini girer
- Harf notuna göre puan hesaplanır
- AKTS ile çarpılarak GANO hesaplanır
- Sonuç ekranda dinamik olarak gösterilir

## Amaç

Bu proje, mobil uygulama geliştirme ve veri yönetimi konularında pratik yapmak amacıyla geliştirilmiştir.
